# cowork-automation

25 AI-powered automation skills for media processing, digital organization, and life admin. Clean up your Downloads folder, extract video clips, auto-tag files, process receipts, and track subscriptions.

## Installation

```bash
# Add the marketplace
/plugin marketplace add fabianaguilar/cowork-automation

# Install the plugin
/plugin install cowork-automation
```

Or test locally:

```bash
claude --plugin-dir /path/to/cowork-automation
```

## Usage

Invoke any skill with:

```

```

For example:
```
/cowork-automation:video-clipper
```
